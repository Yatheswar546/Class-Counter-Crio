# 🔢 Counter App (Class Component)

A React application that implements a counter using a **class-based component**, allowing users to increment and decrement the count value.

---

## 🚀 Live Demo

https://class-counter-crio.netlify.app/

---

## 📌 Features

* Counter starts from **0**
* Increment button increases the count
* Decrement button decreases the count
* Supports negative values
* Built using **React Class Component**
* State management using `this.state` and `this.setState()`

---

## 🛠️ Tech Stack

* React (Vite)
* JavaScript (ES6+)
* CSS

---

## 📂 Project Setup

Run the project locally:

```bash id="k9z1cs"
# Clone repository
git clone <your-repo-link>

# Navigate to project folder
cd class-counter

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## ⚙️ How It Works

1. The component initializes state using `this.state`
2. Clicking **Increment** updates state using `this.setState()`
3. Clicking **Decrement** decreases the count
4. UI updates automatically when state changes

---

## 🧠 Key Concepts

* Class Components in React
* State management using `this.state`
* Updating state with `this.setState()`
* Event handling in class components
